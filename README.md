# Tic-Tac-Toe

A modern Tic-Tac-Toe game built with React where players can play against each other and view the complete move history of the game. The project features an interactive UI, winner detection, and the ability to revisit previous moves.

## 🎮 Features
* Interactive Tic-Tac-Toe gameplay
* Built with React components and hooks
* Move history viewing
* Winner and draw detection
* Restart game functionality
* Beginner-friendly project structure

## 🛠️ Technologies Used
* React
* JavaScript (ES6+)
* CSS3
* Vite

## 🚀 Getting Started
1. Clone the Repository
```
git clone https://github.com/carlbrizuela/tic-tac-toe.git
```
2. Navigate to the Project Folder
```
cd tic-tac-toe
```
3. Install Dependencies
```
npm install
```
4. Start the Development Server
```
npm run dev
```
5. Open in Browser
Visit:
```
http://localhost:5173
```

## 🎯 How to Play
1. Enter player names (Default: Player 1 (X), Player 2 (O)
2. Player X starts first.
3. Players take turns selecting empty squares.
4. The first player to align 3 symbols horizontally, vertically, or diagonally wins.
5. If all squares are filled without a winner, the game ends in a draw.
6. Use the New game button to begin a new game.

## 🕒 Move History
The game stores every move made during gameplay. Players can:
* View all previous moves
* Replay the game step-by-step
* Analyze gameplay decisions
